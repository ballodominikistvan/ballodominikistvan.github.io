NMR Kvantum-számítástechnika: Elméleti Alapok

Bevezetés

Az NMR (Nuclear Magnetic Resonance) kvantumszámítógépek az atommagok spin-állapotait használják információtárolásra. A molekuláris rendszerekben a spinek közötti kölcsönhatások teszik lehetővé a kvantumkapuk megvalósítását.

A Fizikai Modell

Egy külső mágneses térbe helyezett spin Hamilton-operátora leírja a rendszer energiáját és a spinek közötti csatolásokat. Ezek a kölcsönhatások precíz rádiófrekvenciás impulzusokkal manipulálhatók, ami alapvető a kvantumszámítások elvégzéséhez.

Szimuláció Julia nyelven

Az alábbi Julia kód egy egyszerű szimulációt mutat be a Neumann-egyenlet alapján:


using LinearAlgebra

# Paraméterek
dt = 0.01
t_final = 1.0
H = [0.5 0; 0 -0.5] # Hamilton-mátrix

# Időfejlődés operátor: U = exp(-iHt)
U(t) = exp(-im * H * t)

# Kezdeti állapot |0><0|
rho_0 = [1 0; 0 0]

# Állapot t időpillanatban: rho(t) = U*rho_0*U'
rho_t = U(t_final) * rho_0 * U(t_final)'
